# Todo Demo - 使用 yororen-ui 组件的示例应用

这是一个展示 yororen-ui 组件的 Todo 任务管理器演示应用。

## 展示的组件

| 组件 | 用途 |
|------|------|
| `checkbox` | Todo 项的完成状态切换 |
| `switch` | 紧凑/详细模式切换 |
| `tag` | 分类标签显示 |
| `text_input` | 输入新任务标题 |
| `search_input` | 搜索过滤任务 |
| `combo_box` | 选择分类 |
| `modal` | 编辑任务弹窗 |
| `list_item` | 详细模式的任务项 |
| `icon_button` | 编辑和删除按钮 |
| `tooltip` | 按钮悬停提示 |
| `button` | 添加、保存、取消按钮 |
| `heading` | 页面标题 |

## 运行方式

```bash
cd demo/todolist
cargo run
```

## 功能特性

- 添加新任务（支持选择分类）
- 标记任务完成/未完成
- 编辑任务
- 删除任务
- 搜索过滤任务
- 按分类筛选任务
- 紧凑/详细视图模式切换
